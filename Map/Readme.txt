                              #Neighborhood Map project

###Overview
simple project to show how to use APIs and google maps and JS.
##Getting Started
- you can click on the pins on the map to show
  -place name
  -article about it
- you can open the menu from top left of the window
  -search in the given places
-the search will populate its result on the number of pins on the map.

###How to open the app

- download the project file.
- open the folder.
- open index.html file on your browser (make sure you have a good  internet connection).
